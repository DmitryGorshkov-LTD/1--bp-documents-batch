#Обработка массового проведения документов (1С:БСП 3.0)
Назначение:
Внешняя обработка для группового проведения документов "Реализация товаров и услуг" и "Поступление товаров и услуг" в конфигурации 1С:Бухгалтерия предприятия 3.0.
ФУНКЦИОНАЛ
Выбор периода отчета (авточатическая установка времени окончания на 23:59:59)
Загрузка документов за период с реквизитами:
-Дата документа
-Статус проведения (Проведен/Не проведен)
-Сумма документа
Интерактивный выбор документов через чекбоксы
Кнопки быстрого выбора "Выбрать все/Сбросить все"
Сортировка по дате и сумме
Массовое проведение выбранных документов одной кнопкой
ТЕХНОЛОГИИ
Платформа 1С:Предприятие 8.3
Конфигурация: Бухгалтерия предприятие 3.0
Тип: Внешняя обработка (.epf)
Инструменты: Язык запросов 1С, управляемые формы
